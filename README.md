# METALLIC.V1

A small Next.js App Router site for the private METALLIC.V1 ecosystem, including flagship products and a beta access hub.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Notes

- Public assets are served from `/public/uploads`
- The application uses the Next.js App Router and client components for video backgrounds and interactive sections.
